## 🗺️ Lekhika Architecture (STUDY THIS!)

### **AI Management Flow:**

```
1. User Management Page
   ↓
2. Add API Key
   → Stores in `ai_providers` table
   → Fields: (user_id, provider, api_key, name, is_active)
   
3. Select Provider (dropdown of user's keys)
   ↓
4. Fetch Models Button
   → aiModelDiscoveryService.discoverModelsForProvider()
   → Calls OpenAI/Gemini/Claude API to list models
   → Returns: model_id, model_name, context_window, etc.
   
5. Test All Models Button
   → Loops through fetched models
   → Sends test prompt to each
   → If successful: UPSERT to ai_model_metadata (is_active=true)
   → If failed: UPSERT to ai_model_metadata (is_active=false)
   → Updates console logs in real-time
   
6. Active Models Tab
   → Shows ai_model_metadata WHERE is_active=true
   → Inline editing (click cell to edit)
   → Batch operations (activate/deactivate/delete)
   → Pricing scraper
```

### **Database Schema (from Lekhika):**

```sql
-- ai_providers (user's API keys)
CREATE TABLE ai_providers (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES auth.users,
    provider VARCHAR(50), -- 'openai', 'gemini', 'anthropic', etc.
    name VARCHAR(255),    -- User's label (e.g., "OPENAI-PROD-01")
    api_key TEXT,
    description TEXT,
    is_active BOOLEAN DEFAULT true,
    failures INTEGER DEFAULT 0,
    usage_count INTEGER DEFAULT 0,
    last_used TIMESTAMPTZ,
    metadata JSONB,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- ai_model_metadata (discovered models)
CREATE TABLE ai_model_metadata (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES auth.users,
    provider VARCHAR(50),
    model_id VARCHAR(255),        -- e.g., 'gpt-4-turbo'
    model_name VARCHAR(255),      -- Display name
    key_name VARCHAR(255),        -- Which API key discovered this
    key_model VARCHAR(255) UNIQUE, -- Composite key: key_name + model_id
    input_cost_per_million DECIMAL,
    output_cost_per_million DECIMAL,
    context_window_tokens INTEGER,
    tokens_per_page INTEGER,
    specialties TEXT[],
    description TEXT,
    is_active BOOLEAN DEFAULT true,
    last_updated TIMESTAMPTZ,
    metadata JSONB
);
```
