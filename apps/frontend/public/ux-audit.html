The 7 Core Problems

Why a client would feel lost.

1

No Onboarding. No Guide. No Wizard.

Client signs up → lands on dashboard → sees "Total Runs: 0" → has no idea what to do first. No KB created, no brain set up, no context given. Planned in roadmap, never built.

2

Wrong Dashboard. Wrong Metrics.

Dashboard shows "Runs" and "Success Rate". Client cares about: Sends, Opens, Replies, Bookings, Revenue. The /portal page HAS these metrics but it's hidden — not even in the sidebar.

3

Brain Control = Engineer's Panel, Not Client's.

2,048 lines. Memory Palace. Embeddings. Intent Patterns. Query Expansions. Agent Control. No client should ever see this. They should see: "Brain Health: Good. Learning: 47 things. Gaps: 3 things to teach."

4

Dead Buttons Everywhere.

History (brain-chat), Settings (brain-chat), Regenerate (brain-chat), View/Download/Delete (writer), Review Errors (dashboard). Clicking does nothing. This kills trust instantly.

5

Writing Emails = 5 Steps + No Preview.

Writer → New → Pick KB → Pick tone → Pick length → Write prompt → Submit → Go back to list → Can't even see the output. Client should say "Write emails for ICP X" in brain chat and get a preview.

6

Learning Page = Raw Data Dump.

"Dream Logs", "Brain Reflections", "Knowledge Gaps" with raw JSON-like data. Client doesn't know what a "Dream Log" is. Should show: "Your Brain learned 12 new things this week. Here's what it needs from you."

7

Too Many Pages, No Flow Between Them.

11 pages in sidebar. Client doesn't know what "Learning Loop" means. Should be 5 max: Home, Brain, Writer, Knowledge, Performance. Everything else folds into these.

Magic Features — Advanced But Invisible

The client doesn't see the complexity. They see the result.

CMD+K COMMAND PALETTE

⌘K — Do Anything Instantly

Type "write email", "check performance", "teach brain about pricing" — goes straight to the action. Power users love this. New users discover by accident.

SMART NOTIFICATIONS

Brain Nudges

"Your Tech CTO emails got 12% reply rate this week — 3x your average. Want me to write more?" Toast notification that leads to Brain Chat with context pre-loaded.

INLINE TOOL CARDS

Tool Calls → Visual Cards

When Brain uses tools (analyze_signals, get_campaign_insights), show results as beautiful cards in chat — not raw text. Clickable, expandable, actionable.

PROGRESSIVE DISCLOSURE

Start Simple, Grow Complex

New user sees 3 pages. As they use more features, sidebar expands. Brain Control only appears after 30 days or when client asks. Settings > Advanced for power users.

The One Rule

If the client has to think about HOW to use the product, we've failed.

The Brain has 15 tools, 3 tiers, 7 prompt layers, a hallucination interceptor, and a dream cycle. The client should never know any of that exists. They should open the app, see their numbers, talk to their Brain, and watch emails go out. That's it.
