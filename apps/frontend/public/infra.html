02 — satellite fleet

5 Disposable VPS Boxes

Each satellite is a separate SMTP server ($50/mo) from a trusted vendor. Each is optimized for 30K emails/day with its own dedicated IP. Tommy's model: cap each mailbox at 3K/day × 10 mailboxes = 30K/day per box (exactly the box's rated capacity). If one gets blacklisted, you lose $50 — not your business.

WHY SEPARATE VPS, NOT SAME-BOX IPS?
Subnet fingerprinting: Gmail sees that 107.172.56.68, .69, .70 are all in the same /28 block. Same ASN, same datacenter. Red flag.
Collateral damage: One blacklisted IP can taint the entire /28 subnet.
Single point of failure: Server down = ALL satellites down.
Disposability: A $50/mo SMTP box is replaceable in hours. Your $90/mo command center with all your data is not.

08 — dns mastery

DNS Configuration

Every satellite domain needs these 5 records. PTR is set at the satellite VPS provider's dashboard — NOT in Cloudflare.

1. SPF

Like a guest list. SPF tells Gmail: "Only this satellite VPS IP is allowed to send for this domain."
Authorize the satellite VPS IP — NOT the command center IP.
Type: TXT  |  Name: @
v=spf1 mx ip4:{satellite_vps_ip} ~all

2. DKIM

A wax seal on every letter. Gmail checks if it's intact. If tampered, the seal breaks.
Generated by OpenDKIM on each satellite VPS. Each domain has its own key pair.
Type: TXT  |  Name: mail._domainkey
v=DKIM1; k=rsa; p=MIIBIjANBgkqhkiG9w...

Generated on satellite VPS: opendkim-genkey -s mail -d domain.com

3. DMARC

The school principal. If SPF or DKIM fails, DMARC decides: quarantine or reject.
Type: TXT  |  Name: _dmarc
v=DMARC1; p=quarantine; pct=100; aspf=r; adkim=r;

4. PTR / rDNS

Gmail asks: "Does this satellite IP really belong to mail.domain.com?" PTR is the background check.
Set at the satellite VPS provider's dashboard — each provider has their own rDNS panel.
{satellite_vps_ip}  ──►  mail.{domain}.com

Set in: RackNerd VPS panel / BuyVM panel / Contabo panel

5. MX Record

The mailing address for replies. Tells the internet where to deliver incoming mail.
Type: MX  |  Name: @  |  Priority: 10
Value: mail.{domain}.com
